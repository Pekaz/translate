# What if Instagram Got Rid of Likes?
### 인스타그램이 "좋아요"를 없앤다면?

공개적 지표에 기반을 둔 소셜 네트워크들이 이제는 그걸 숨기기 시작했다.

By John Herrman

May 31, 2019

```

몇가지 업데이트를 발표했습니다. 스토리를 포스팅하는 새로운 방식, 물건을 사는 새로운 방식, 카메라
```
